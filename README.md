# ✊✋✌️ Rock Paper Scissors Game

An interactive **Rock Paper Scissors** game built using **HTML**, **CSS**, and **JavaScript**. Challenge the computer in this classic game while enjoying a clean, responsive, and engaging user interface.

This project demonstrates the fundamentals of front-end web development by implementing game logic, dynamic UI updates, score tracking, and event-driven programming using vanilla JavaScript.

---

## 🚀 Features

* ✊ Choose Rock, Paper, or Scissors
* 🤖 Play against a computer with randomly generated moves
* ⚡ Instant game result after every round
* 📊 Real-time score tracking
* 🎨 Clean and responsive user interface
* 📱 Mobile-friendly design
* 🔄 Restart and play unlimited rounds
* 💻 Built entirely with HTML, CSS, and JavaScript

---

## ▶️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/rock-paper-scissors.git
```

### Run the Project

1. Open the project folder.
2. Double-click **index.html** or open it in your preferred web browser.
3. Start playing!

No installation or additional dependencies are required.

---

## 🎮 How to Play

1. Select **Rock**, **Paper**, or **Scissors**.
2. The computer automatically makes its choice.
3. The winner is determined according to the classic rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. Scores are updated after each round.
5. Continue playing as many rounds as you like.
